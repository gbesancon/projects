# Pronouns

## Personal

- Εγώ -> I
- Εσύ -> You (singular)
- Αυτός / Αυτή / Αυτό -> He / She / It
- Εμείσ -> We
- Εσείσ -> You (plural / formal)
- Αυτοί / Αυτές / Αυτά -> They (male / female / neutral)

## Possessive

- Μου -> My
- Σου -> Yours (singular)
- Του / Της / Του -> His / Hers / Its (neutral)
- Μας -> Ours
- Σας -> Yours (plural / formal)
- Τους -> Theirs
