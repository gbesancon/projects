# Greetings
- Γειά σου / Γειά σας -> Hi
- Καλημέρα / Καλημέρα σας -> Good morning
- Καλησπέρα / Καλησπέρα σας -> Good afternoon / evening
- Καληνύχτα / Καληνύχτα σας -> Good night
- Παρακαλώ -> please, welcome
- Ευχαριστώ -> thank you
