# Pronouns
- Εγώ -> I
- Εσύ -> You

# Questions
- Ποιός / Ποιοί -> Who
- Αυτός / Αυτοί -> This / These
- Πόσοι / Πόσες / Πόσα -> How many
- Πόσο -> How much
- Πού -> Where
- Πώς -> What
- Τι -> What
- Γιατί -> Why, because
  
# Articles
- Δεν -> not (goes in front of the verb to make it negative)
- Θα -> will
- Στο -> at the, on the, in the

# Verbs
- Είμαι -> to be
- Θέλω -> to want
- Κάνω -> to do
- Έχω -> to have
- Μένω -> to stay, to live
- Ξέρω -> to know
- Καταλαβαίνω -> to understand
- Με λένε -> I am called, my name is
- Άργησα -> I am late

# Conjugate: to be
- Είμαι -> I am
- Είσαι -> you are
- Είναι -> he, she, it is
- Είμαστε -> we are
- Είστε, είσαστε -> you all are
- Είναι -> they are

# Conjugate: to stay
- Μένω -> I stay
- Μένεις -> you stay
- Μένει -> he, she, it stay
- Μένουμε -> we stay
- Μένετε -> you all stay
- Μένουν -> they stay
